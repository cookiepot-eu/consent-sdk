# Changelog

All notable changes to the CookiePot Consent SDK will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial public release
- Core consent management functionality
- React integration with hooks and components
- Google Consent Mode v2 support
- Script auto-blocking
- Multi-language support (EN, NL, DE, FR, ES, IT, PT, PL)
- Customizable banner theming
- TypeScript support
- Offline-first design

## [0.2.0] - 2026-01-29

### Changed
- Fixed package.json exports order for proper TypeScript resolution
- Added repository directory field for monorepo support
- Added bugs field for issue tracking

### Fixed
- Build warnings about "types" condition placement

## [0.1.0] - 2026-01-28

### Added
- Initial SDK implementation
- Vanilla JavaScript API
- React integration
- Basic consent banner
- Local storage persistence
- Event system
- API client for CookiePot backend
